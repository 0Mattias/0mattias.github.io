WhalEmu

a real-mode x86 emulator with a step debugger · runs entirely in your browser

Real mode is the 16-bit world an x86 processor wakes up in: eight registers, four flags, a stack, no protections. WhalEmu emulates it with a step debugger so you can watch each instruction land. The full build does real instruction encoding and stays private; this page is a working slice with a two-pass assembler and two dozen instructions.

Fibonacci is already loaded and assembled below. Press Step and watch AX and BX trade values while CX counts the loop down. The highlighted line is the next instruction to run; Run plays it through on its own.

Pick a sample or write your own, then Assemble.

    Registers AX BX CX DX SI DI BP SP, all 16-bit. Flags ZF SF CF OF. Immediates in decimal, 0x hex, or trailing-h hex. Labels end with a colon, comments start with a semicolon. OUT prints a register to the output panel, standing in for the DOS print interrupt. The listing executes by index rather than from encoded bytes; the full emulator does real instruction encoding, this page keeps the part that teaches.
