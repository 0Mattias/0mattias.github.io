bettermemory

Verification-grade memory for AI coding agents. Retrieved on demand, with three staleness signals on every hit.

Local MCP server (and Claude Code plugin) that fixes the failure mode every other LLM-memory tool ships with: auto-injecting stored facts into every conversation. Retrieval is opt-in, and every hit carries a staleness verdict (calendar age, filesystem path drift, and git commit drift), so the model knows when a fact has rotted before it leans on it. Claims about you stage for confirmation, never silently. Past 2.0 it grew an episodes tier for multi-agent and loop run-state, a silent-miss probe and cold-endorsement health surface, and opt-in write-reflex capture. Markdown plus YAML on disk, so you can grep it, git log it, hand-edit it. Works with Claude Code, Cursor, Cline, and any MCP client.
