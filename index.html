bettermemory

Currently building · v3.10 · updated June 11, 2026

Storing facts is the easy half of agent memory; knowing a fact has held up weeks and forty commits later is the hard half. bettermemory is built around the hard half: every memory it serves carries a verdict on whether the fact still holds.
