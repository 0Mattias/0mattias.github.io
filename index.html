Anaphase

Anaphase

Private

A covert-channel AI agent that runs over whatever protocol still gets out.

A pluggable transport layer hides agent traffic in DNS (UDP, DoT, DoH), ICMP echo, or NTP extension fields — whatever the network in front of you doesn't block. AES-256-GCM on the payload, HKDF-SHA256 per-session keys. The agent itself runs shell commands, edits files, searches code, and fetches URLs, gated by approval prompts on anything destructive. Multi-turn sessions across Anthropic, Gemini, OpenAI, and OpenRouter. Pure C, no runtime dependencies.

About

About

Most of what's on this page was written this spring. I pick the language each project actually needs — C for transport-level work, Rust where async and memory safety matter, Java for game internals, Go where Postgres needs a fast feeder, TypeScript when there has to be a UI. The stack follows the problem.

I run my own infrastructure on a three-node Tailscale homelab — Forgejo, a Sliver C2, Velociraptor for DFIR, the usual self-hosted stack — so the security work has somewhere real to live.
